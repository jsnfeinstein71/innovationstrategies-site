# Innovation Strategies Site

A simple, static Next.js landing page for Innovation Strategies LLC.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production Build

Create a production build:

```bash
npm run build
```

Run the production server locally:

```bash
npm run start
```

## Deployment

This site is static and can be deployed to any platform that supports Next.js,
including Vercel, Netlify, or a Node-compatible host.

For Vercel:

1. Connect this repository to Vercel.
2. Use the default Next.js build settings.
3. Set the production domain to `innovationstrategies.pro`.

No environment variables are required.
